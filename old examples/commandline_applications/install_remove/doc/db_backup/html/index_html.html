    <h1 id=“label-%60db_backup%60+-+Iteration-aware+MySQL+database+backups”>`db_backup` - Iteration-aware MySQL database backups<span><a href=“#label-%60db_backup%60+-+Iteration-aware+MySQL+database+backups”>&para;</a> <a href=“#documentation”>&uarr;</a></span></h1> <dl class=“rdoc-list note-list”><dt>Author <dd> <p>David Copeland (<a href=“dave@example.com)dave@example.com”>dave@example.com)> </dd><dt>Copyright <dd> <p>Copyright © 2011 by David Copeland</p> </dd><dt>License <dd> <p>Distributes under the Apache License, see LICENSE.txt in the source distro</p> </dd></dl>

    <p>This application provides an easy interface to backing up MySQL databases, using a canonical naming scheme for both daily backups and “end-of-iteration” backups.</p>

    <h2 id=“label-Install”>Install<span><a href=“#label-Install”>&para;</a> <a href=“#documentation”>&uarr;</a></span></h2>

    <p>Install:</p>

    <pre>gem install db_backup</pre>

    <h2 id=“label-Use”>Use<span><a href=“#label-Use”>&para;</a> <a href=“#documentation”>&uarr;</a></span></h2>

    <p>Backup a database:</p>

    <pre>db_backup.rb my_database</pre>

    <p>For more help:</p>

    <pre>db_backup.rb –help gem man db_backup.rb</pre>

    <h2 id=“label-Developing+for+%60db_backup%60”>Developing for `db_backup`<span><a href=“#label-Developing+for+%60db_backup%60”>&para;</a> <a href=“#documentation”>&uarr;</a></span></h2>

    <p>First, install bundler:</p>

    <pre>gem install bundler</pre>

    <p>Get the development dependencies</p>

    <pre>bundle install</pre>

    <p>Most of the code is in `bin/db_backup.rb`.</p> </div>
