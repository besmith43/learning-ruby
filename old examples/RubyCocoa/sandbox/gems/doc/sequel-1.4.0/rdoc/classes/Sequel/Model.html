set_primary_key(*key)

Sets primary key, regular and composite are possible.

Example:

  class Tagging < Sequel::Model
    # composite key
    set_primary_key :taggable_id, :tag_id
  end

  class Person < Sequel::Model
    # regular key
    set_primary_key :person_id
  end

You can even set it to nil!
